Cloudflare allows you to apply several changes to your zone records at the same time. You can use the Dashboard to delete or edit DNS records in bulk, or perform further batched operations via API.

## Availability and limits

Batch DNS record changes is available on all plans.

The number of records that you can operate with in one action depends on your zone plan:

- Free: 200
- Pro: 3,500
- Businees: 3,500
- Enterprise: 3,500

## Use the API

### Requirements

Token permissions, familiarity with API (Patch vs Put)

### Aspects to consider

- Execution order
- Propagation is not atomic

### Example request

```bash
{
    "deletes": [
        {
            "id": "143004ef463b464a504bde5a5be9f94a"
        },
        {
            "id": "243004ef463b464a504bde5a5be9f94a"
        }
    ],
    "patches": [
        {
            "id": "343004ef463b464a504bde5a5be9f94a",
            "content": "192.0.2.45"
        },
        {
            "id": "443004ef463b464a504bde5a5be9f94a",
            "name": "update.example.com",
            "proxied": true
        }
    ],
    "puts": [
        {
            "id": "543004ef463b464a504bde5a5be9f94a",
            "content": "192.0.2.50",
            "name": "no-change.example.com",
            "proxied": false,
            "ttl:": 1
        },
        {
            "id": "643004ef463b464a504bde5a5be9f94a",
            "content": "192.0.2.45",
            "name": "no-change.example.com",
            "proxied": false,
            "ttl:": 3000
        }
    ],
    "posts": [
        {
            "name": "@",
            "type": "A",
            "content": "192.0.2.45",
            "proxied": false,
            "ttl": 3000
        },
        {
            "name": "a.example.com",
            "type": "A",
            "content": "192.0.2.45",
            "proxied": true
        }
    ]
}
```
